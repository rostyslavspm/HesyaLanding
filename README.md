# Hesya Landing Page

Landing page for [Hesya](https://hesya.app) — a quiet pause app for iOS.

## Stack

- Next.js 16 (App Router) + TypeScript
- Tailwind CSS v4
- Framer Motion
- Deployed on Vercel

## Pages

- `/` — Marketing homepage
- `/privacy` — Privacy policy
- `/support` — Support & FAQ

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Screenshots

Drop app screenshots into `public/screenshots/` — the phone frame component will display them automatically once added.
